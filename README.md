# Simple E-commerce Store

## Full Stack Development Internship - CodeAlpha

### Project Overview

A simple e-commerce web application developed using Django.

### Features

* User Registration
* User Login
* Product Listings
* Product Detail Page
* Shopping Cart
* Order Processing
* Database Integration

### Technologies Used

* Python
* Django
* HTML
* CSS
* JavaScript
* SQLite

### How to Run

1. Install Django
2. Run:
   python manage.py migrate
3. Start the server:
   python manage.py runserver
4. Open:
   http://127.0.0.1:8000/
